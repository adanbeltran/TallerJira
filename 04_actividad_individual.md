## Problema

La universidad necesita una solución para atender situaciones de:

- bullying;
- ciberacoso;
- conflictos de convivencia.

Muchos estudiantes no saben cómo pedir ayuda, temen represalias o abandonan el proceso por falta de seguimiento.

## Usuarios del sistema
- estudiante solicitante;
- estudiante testigo;
- orientador;
- coordinador;
- administrador.

## Flujo básico del PMV
1. el estudiante solicita apoyo;
2. el sistema crea el caso;
3. el orientador revisa el caso;
4. el orientador clasifica la urgencia;
5. se realiza el seguimiento;
6. el caso puede cerrarse.

## Reglas básicas
- si el reporte es anónimo, no hay seguimiento personalizado;
- si es identificado, debe existir información mínima;
- solo Bienestar accede a datos sensibles;
- el caso debe manejar estados;
- debe existir clasificación por urgencia.
